# 高考名句名篇默写题目汇编（按年份排序）

> 整理说明：本表格由近及远排列，涵盖了高考中出现的经典默写题目。

## 📅 2025年

### 北京卷

**题目：** 【选择性必修上·论语】 孔子认为，君子应该“文质彬彬”，如果处理不好文质关系，就会产生两种结果：“{1}，{2}。”

**答案：** `质胜文则野,文胜质则史`

---

### 全国二卷

**题目：** 【必修上·赤壁赋】 周末的音乐会圆满成功，媒体报道时引用苏轼《赤壁赋》中的“{1}，{2}”两句，形容洞箫的尾音婉转悠长，连绵不断。

**答案：** `余音袅袅,不绝如缕`

---

### 全国二卷

**题目：** 【选择性必修下·临安春雨初霁】 老年大学的教室里悬挂着一副对联，写的是陆游《临安春雨初霁》中的“{1}，{2}”，对应着开设的书法和茶艺两门课。

**答案：** `矮纸斜行闲作草,晴窗细乳戏分茶`

---

### 全国一卷

**题目：** 【必修上·师说】 在毕业典礼上，柳教授谦逊地引用韩愈《师说》中的“{1}，{2}”两句，勉励学生们青出于蓝而胜于蓝。

**答案：** `(是故)弟子不必不如师,师不必贤于弟子`

---

### 全国一卷

**题目：** 【必修上·鹊桥仙】 张老师给守卫边疆的丈夫发信息，引用秦观《鹊桥仙》中“{1}，{2}”两句，表示虽不能朝夕相守，但仍情长意久。

**答案：** `两情若是久长时,又岂在朝朝暮暮`

---

**题目：** 【必修上·永遇乐·京口北固亭怀古】 在《永遇乐（千古江山）》中，辛弃疾回顾了元嘉年间的那次北伐，宋文帝刘义隆本希望能够“{1}”，但是由于行事草率，最终却“{2}”。

**答案：** `封狼居胥,赢得仓皇北顾`

### 新课标Ⅰ卷

**题目：** 【必修上·劝学】 《荀子·劝学》指出，蚯蚓虽然身体柔弱，却能“{1}，{2}”，是用心专一的缘故。

**答案：** `上食埃土,下饮黄泉`

### 新课标Ⅰ卷

**题目：** 【未收录·其他/未收录】 在《出师表》开头，诸葛亮向后主指出，先帝刘备过早去世，“{1}，{2}”，正是危急存亡之时。

**答案：** `今天下三分,益州疲弊`

### 山东卷

**题目：** 【未收录·逍遥游】 《庄子·逍遥游》描写迁徙南溟的大鹏击水之广、飞升之高的句子是“{1}，{2}。”

**答案：** `水击三千里,抟扶摇而上者九万里`

### 山东卷

**题目：** 【必修上·虞美人】 李煜《虞美人》词中“{1}，{2}”两句，以江水比喻自己的愁绪，抒发亡国之君无尽的痛苦之情。

**答案：** `问君能有几多愁,恰似一江春水向东流`
